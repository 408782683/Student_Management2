# fronted - Vue 3 前端示例

使用 Vite + Vue 3 搭建的简单前端，便于对接后端接口（端口 8080）。

## 开发
```bash
cd fronted
npm install
npm run dev
```

Vite 代理已配置 `/api` -> `http://localhost:8080`，无需额外跨域配置。

## 页面
- `/` 登录（获取验证码并调用登录接口）
- `/admin` 管理员：师生信息增删改查、学院/专业/班级管理、课程/培养计划、教学任务分配、课表上传
- `/teacher` 教师：修改密码、教学任务查询、成绩录入/查询、课表查看、点名册导出
- `/student` 学生：修改密码、成绩查询/打印、课表查看、自主选课

可按课程作业需求自行扩展样式、表单和数据校验。
